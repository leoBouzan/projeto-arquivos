# FileShare Web

Este diretorio contem o frontend Angular do projeto **FileShare**, desenvolvido como **projeto academico** no contexto de **Aplicacoes para Web**.

O objetivo deste frontend e servir como interface para os fluxos principais do sistema, como upload de arquivos, consulta de disponibilidade e download por link temporario. Ele **nao representa um produto final** e faz parte de um trabalho de estudo e arquitetura.

## Execucao local

Na raiz do repositorio, o frontend pode ser iniciado com:

```bash
./scripts/run-web.sh
```

Se preferir executar manualmente:

```bash
cd src/frontend/fileshare-web
npm start
```

O projeto sobe em:

```text
http://localhost:4200
```

## Build

```bash
npm run build
```
